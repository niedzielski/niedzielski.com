I work for Wikipedia. Here are some projects I've made meaningful contributions to:

  • Wikimedia Vue User Interface component library: Technically, a fine Vue.js library for Wikipedia, MediaWiki, and beyond. Socially, an amazing journey for Wikimedia.
  • Page Previews: Intelligent summary dialogs shown when hovering over Wikipedia article links.
  • Desktop Improvements Project: An overhaul of the Wikipedia user interface.
  • Wikipedia for Android and the Node.js backend service: Wikipedia's native Android application. The backend service serves both data and cross-platform (Android / iOS) WebView presentation.
  • Marvin: An isomorphic TypeScript Preact web app version of Wikipedia.
  • Wikipedia Mobile: The Wikipedia mobile site.
